About this course

Program's goal

Train programmers in the technologies most used in the development of Websites and mobile applications.

2 years

61 credits

6 semesters

Internship

Prerequisites

International applicants must at least hold a high school degree or equivalent, and must hold an English test (ex. IELTS, TOEFL, etc.) and meet our grade requirements.

 

Canadian students must hold a minimum of a Secondary 5 diploma with strong mathematics (TS or SN) or equivalent education.

Professional perspectives

Full-stack web developer

Back-end developer

Front-end developer

Android Mobile Developer

Software developer

Analyst programmer

Technologies taught

C, C++, C#, SQL, Java, Android JavaScript, Node.js, Angular JS, React, PHP, HTML5, CSS3, Bootstrap and more.

Courses Details

Semester 1
Technical and programming logic

Become familiar with different working techniques and algorithmic logic used in the programming environment. This course allows you to acquire the basic knowledge necessary for the job of programmer analyst.

HTML Web Programming

Create the structure of web pages using HTML, and enhance them with style sheets (CSS).

Introduction to structured programming (C)

Learn the concepts, techniques and a structured programming methodology with C.

Introduction to object-oriented programming (C++)

Learn the basic concepts of object-oriented programming: classes objects, inheritance, polymorphism, etc..

Semester 2
Object-oriented programming I (Java I)

Address graphical application programming and event management, threads and socket communication.

Programming in a graphical environment (C#)

Evolve standard Windows applications with C# in the environment to become familiar with problem analysis, concepts and terms of event programming.

Database

Appropriate the principles of modeling databases. Master the Transact-SQL database programming language.

Object-oriented programming (Java II)

Deepen the knowledge of Java and master the server-side development  (J2EE, JSP, Servlet). Know how to apply an MVC architecture.

Semester 3
Web programming I (Advance HTML)

Learn advanced HTML5 and CSS3 techniques to maximize browser compatibility, production time and web page performance. Master notions and concepts of accessibility and user experience. Design and develop a web application in a graphical environment. Introduction to Bootstrap.

Web programming II (JavaScript)

Dynamize web pages with JavaScript, animate them and make them interactive by adding script and using libraries like jQuery.

Web programming III (PHP)

Learn about Web programming and HTTP protocol.

Software structure (Framework)

Become familiar with sine frameworks using advanced server-side development technologies such as React, Node.js, Angular JS.

Semester 4
Mobile application development (Android)

Know the fundamentals and peculiarities of programming for mobile devices in general. Learn how to develop applications for the Android system, understand the concepts specific to this environment, and learn how to harness the data, services, and peripherals of mobile devices. Explore the advanced features of the Android system.

Deployment and software integration

This course covers application deployment in the production environment and various tools that are used for this purpose. Continuous integration, the management of repositories and the administration of application servers are discussed.

Analysis project

Become familiar with a method of analysis and object oriented modeling. Plan the development of the project.

Production project

Create a complete web application that will integrate all the concepts seen.

Semester 5 & 6
Dynamic job search method

Become familiar with the job market. Produce CV. Develop job search techniques.

Design and realization of the portfolio

Assemble the different projects produced during the lessons.

Intership I and II

Consolidate the notions acquired by a concrete application of concepts and learning in real situations.
